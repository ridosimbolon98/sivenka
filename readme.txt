1. Install XAMPP
2. Setting Environment di Path dengan nilai=> c:\xampp\php
3. Pindahkan Folder Aplikasi ke folder c:\xampp\httdocs\
4. Jalakan xampp-control Apache dan MySQL
5. Buka Browser, ketik alamat: localhost/phpmyadmin
6. Create database dengan nama: simlar
7. Import database dengan file simlar.sql yang ada di folder database
8. Install driver printer POS58 -> Pastikan sharing printer on/ceklist
9. Silakan jalankan aplikasi dengan ketik address: localhost/simlar
